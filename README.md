# PROYECTO HACKATON 2026

## PROYECTO "ASISTENTE DE CARRETERA AFN"

Buen día, sean bienvenidos al repositorio de nuestro **proyecto "asistente de carretera AFN"**.

Este documento sirve como guía para entender, estructurar y visualizar el proyecto **ASISTENTE DE CARRETERA AFN**, una interfaz web operativa diseñada para simular la interacción con un agente de IA especializado en soporte técnico logístico.

---

## VISIÓN DEL PROYECTO

Este proyecto consiste en una **página web estática y responsiva**, creada con el objetivo de aplicar conceptos aprendidos en el curso sobre **maquetación con HTML y diseño con CSS**.

La página presenta al **Asistente de Carretera AFN**, un agente de IA diseñado para apoyar a operarios a dar solución a fallas mecánicas básicas en carretera, con el fin de disminuir el tiempo de respuesta. El sitio incluye:

- **Formulario de Registro:** Captura de datos críticos (tipo de falla, ruta y descripción).
- **Panel de Diagnóstico IA:** Resultados de clasificación, prioridad y acciones sugeridas.
- **Jerarquía de Prioridades:** Identificación visual de niveles de urgencia (Alta, Media, Baja).
- **Protocolo Operativo:** Listado de pasos técnicos y acciones automáticas de seguridad.

A diferencia de proyectos iniciales, en esta etapa se priorizó una **estética industrial y profesional**, logrando una experiencia de usuario (UX) enfocada en la eficiencia y la claridad bajo condiciones de operación real.

El enfoque principal sigue siendo el aprendizaje, aplicando buenas prácticas como:
- Uso estricto de etiquetas semánticas.
- Separación de estructura (HTML) y estilos (CSS) en archivos independientes.
- Uso de **CSS Grid** y **Flexbox**.
- Diseño responsivo.

---

## TECNOLOGÍAS UTILIZADAS

### HTML5
Utilizado para la estructura del tablero operativo, aplicando etiquetas semánticas como:
- `header`, `main`, `aside`, `section`, `form`.
- `h1`, `h2`, `select`, `textarea`, `button`.

### CSS
Utilizado para el diseño visual y la responsividad del sitio:
- **Variables CSS (`:root`):** Para una paleta de colores corporativa consistente.
- **Flexbox & Grid:** Para la distribución de paneles de control.
- **Efectos de Estado:** Animaciones de "pulso" para indicadores en vivo y transiciones en botones.
- **Media Queries:** Adaptabilidad total a dispositivos móviles y tablets.

> **Nota:** El proyecto **no utiliza JavaScript**
